# HonkaiLab

E commerce app build with flutter framework using provider as state management and MVVM pattern

## Features

- [x] Login
- [x] Register
- [x] Product
- [x] Category Product
- [x] Review
- [x] Wishlist
- [x] Cart
- [x] Transaction
- [x] Profile
